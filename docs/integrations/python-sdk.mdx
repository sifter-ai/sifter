# Python SDK

The `sifter-ai` SDK is a pure HTTP client that connects to a running Sifter server (cloud or self-hosted).

## Install

```bash
pip install sifter-ai
```

## Client

```python
from sifter import Sifter

s = Sifter(
    api_url="https://sifter.run",      # or http://localhost:8000 for self-hosted
    api_key="sk-...",                   # or set SIFTER_API_KEY env var
)
```

## Error handling

The SDK exposes a unified surface across OSS and Cloud. Cloud-only methods (connectors, billing) raise specific errors when the backend doesn't support them or when your plan doesn't include them.

```python
from sifter import Sifter, NotAvailableError, PlanLimitError

s = Sifter(api_url="...", api_key="sk-...")

try:
    conn = s.connectors.gdrive.list()
except NotAvailableError:
    # Backend is OSS/self-hosted — cloud-only feature not available here.
    print("Connectors require Sifter Cloud.")
except PlanLimitError as e:
    # Cloud backend, but the current plan doesn't allow this operation.
    print(f"Upgrade required: {e.required_plan}")
```

| Exception | When it fires |
|-----------|---------------|
| `NotAvailableError` | Called a Cloud-only method against an OSS backend (endpoint returns 404/501). |
| `PlanLimitError` | Cloud backend returned `402 Payment Required` — current plan doesn't include the feature or the call exceeds the plan quota. Has `required_plan` and `limit` attributes. |
| `APIError` | Generic 4xx/5xx with message from the server. |
| `AuthError` | 401/403 — check your API key. |

<Info>
Autocomplete surfaces every method on both OSS and Cloud. The runtime decides if the call succeeds — so you can write one codebase that targets either backend.
</Info>

## Sifts

```python
# Create
sift = s.create_sift("Invoices", "client, date, total amount, VAT number")

# Read — one page
page = s.list_sifts(limit=50, offset=0)   # returns Page(items, total, limit, offset)
print(f"{page.total} sifts total")

# Read — iterate all
for sift_data in s.iter_sifts():
    print(sift_data["name"])

# Update / delete
sift = s.get_sift("sift-id")
sift.update(name="Invoices 2024")
sift.delete()
```

## Documents and results

```python
sift.upload("./invoices/")           # file or directory
sift.wait(poll_interval=2, timeout=300)  # block until done

# One page of records
page = sift.records(limit=100, offset=0)   # Page(items, total, limit, offset, next_cursor)

# Iterate all records across pages (recommended for large datasets)
for record in sift.iter_records():
    print(record["extracted_data"])

results = sift.query("Total by client")
sift.export_csv("invoices.csv")
```

## Folders

```python
folder = s.create_folder("Q1 Contracts")
folder.add_sift(sift)
folder.upload("./contracts/q1/")

# One page
page = folder.documents(limit=50, offset=0)

# Iterate all
for doc in folder.iter_documents():
    print(doc["filename"])

folder.sifts()          # Page of linked sifts
folder.iter_sifts()     # iterate all linked sifts
folder.delete()
```

## Multi-sift pipeline

```python
parties = s.create_sift("Parties", "contracting parties, dates, signatories")
clauses = s.create_sift("Clauses", "non-compete, termination conditions")

folder = s.create_folder("Contracts 2024")
folder.add_sift(parties)
folder.add_sift(clauses)
folder.upload("./contracts/")

parties.wait()
print(parties.records())
```

## One-liner

```python
# Create temp sift, upload, wait, return records
records = s.sift("./invoices/", "client, date, total")
```

## Event callbacks

Register local callbacks for events on a sift or folder. Callbacks fire during `wait()`.

```python
sift.on("sift.document.processed", lambda doc_id, record: print(record))
sift.on("sift.error", lambda doc_id, error: print(f"Error: {error}"))
sift.on("sift.*", lambda *args: print(args))  # wildcard

folder.on("folder.document.uploaded", lambda doc: print(f"New: {doc}"))
```

Wildcard patterns: `*` matches any single segment, `**` matches any number.

## Webhooks

Register server-side HTTP callbacks:

```python
s.register_hook(
    events=["sift.document.processed", "sift.error"],
    url="https://example.com/webhook",
    sift_id=sift.id,  # optional: scope to one sift
)
hooks = s.list_hooks()
s.delete_hook(hook_id)
```

See [Webhooks Reference](/integrations/webhooks-reference) for payload schemas.
