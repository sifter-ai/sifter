# Pricing

Sifter is **open source and free forever** under Apache 2.0 — self-host on your own infrastructure, bring your own LLM key, pay for nothing.

Sifter **Cloud** is the hosted version: managed infra, remote MCP endpoint, Google Drive + mail-to-upload ingress, billing, SSO, audit log, share links.

<Info>
**1 document = 1 file up to 10 pages.** PDFs longer than 10 pages are rejected at upload — split and retry. Free plans cap at 5 MB per file; paid plans start at 25 MB and scale up.
</Info>

## What's in every tier

Chat over your documents, spec-driven [dashboards](/concepts/dashboards), [webhooks](/concepts/webhooks) with HMAC signing + retry, natural-language queries, aggregations, and folder organization — all OSS, all included on Free. Paying unlocks volume, API/MCP access, connectors, and managed operations.

<Info>
**MCP remote** is the flagship Cloud upgrade. Free keys receive `402 Payment Required` on any tool call — upgrade to Starter to unlock the managed `https://sifter.run/mcp` endpoint that any MCP-aware client (Claude, ChatGPT, Gemini, Cursor) can hit with an `Authorization: Bearer` header.
</Info>

## Self-hosting

The OSS engine is free forever. Run everything — chat, dashboards, webhooks, SDK, MCP stdio — on your own infra with your own LLM key.

```bash
git clone https://github.com/bfortunato/sifter
cd sifter/code
cp .env.example .env  # set SIFTER_LLM_API_KEY
docker compose up -d
```

The 10-page cap is enforced by default (`SIFTER_MAX_PDF_PAGES=10`) and can be raised in self-hosted deployments. See [Self-hosting](/self-hosting/docker-compose) for the full setup.
