## Self-hosting

The OSS engine is **free forever** under Apache 2.0. Run it on your own infrastructure — no document limits, no usage fees. You pay only for your own LLM API key.

```bash
git clone https://github.com/bfortunato/sifter
cd sifter/code
cp .env.example .env  # set SIFTER_LLM_API_KEY
docker compose up -d
```

The 10-page cap applies by default (`SIFTER_MAX_PDF_PAGES=10`) and can be raised in self-hosted deployments.

See [Self-hosting](/self-hosting/docker-compose) for the full setup guide.
