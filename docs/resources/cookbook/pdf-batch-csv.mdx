# Document batch → CSV

A minimal generic pattern: give Sifter a folder of documents and a description of what to extract, get a CSV back. Works with PDFs, Word, Excel, PowerPoint, and images.

## Code

```python
from sifter import Sifter

def batch_extract(folder_path: str, instructions: str, output_csv: str) -> None:
    s = Sifter(api_key="sk-...")
    sift = s.create_sift(name="Batch extraction", instructions=instructions)
    sift.upload(folder_path).wait()
    sift.export_csv(output_csv)
    print(f"Exported {sift.records().total} records to {output_csv}")

batch_extract(
    folder_path="./documents/",
    instructions="Extract: title, author, date, summary (one sentence)",
    output_csv="output.csv",
)
```

## TypeScript

```ts
import { SifterClient } from "@sifter-ai/sdk";
import { writeFileSync } from "fs";

const sifter = new SifterClient({ apiKey: "sk-..." });
const sift = await sifter.createSift("Batch extraction", "title, author, date, summary");
await (await sifter.uploadToSift(sift.id, ["./documents/"])).wait();
const csv = await sift.exportCsv();
writeFileSync("output.csv", csv);
```

## One-liner (Python)

```python
from sifter import Sifter

records = Sifter(api_key="sk-...").sift("./documents/", "title, author, date, summary")
print(records)
```
