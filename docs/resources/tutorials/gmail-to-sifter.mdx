# Gmail attachments → Sifter

Automatically extract data from PDF attachments you receive by email — no manual uploads.

## Option A: Zapier

1. Create a Zap: **Gmail → New Attachment** trigger
2. Add a **Webhooks by Zapier** action: POST to `https://api.sifter.ai/api/folders/{folder_id}/documents`
3. Set headers: `X-API-Key: sk-...`
4. Map the attachment file to the `file` form field

Every matching email attachment is now uploaded to your Sifter folder and processed automatically.

## Option B: Google Apps Script

```javascript
function checkGmail() {
  const FOLDER_ID = "your-folder-id";
  const API_KEY = "sk-...";
  const LABEL = "sifter-inbox"; // create this label in Gmail

  const threads = GmailApp.search(`label:${LABEL} is:unread`);
  threads.forEach(thread => {
    thread.getMessages().forEach(msg => {
      msg.getAttachments().forEach(att => {
        if (att.getContentType() === "application/pdf") {
          const blob = att.copyBlob();
          UrlFetchApp.fetch(
            `https://api.sifter.ai/api/folders/${FOLDER_ID}/documents`,
            {
              method: "POST",
              headers: { "X-API-Key": API_KEY },
              payload: { file: blob },
            }
          );
        }
      });
      msg.markRead();
    });
  });
}
```

Set a time-based trigger to run `checkGmail` every 5 minutes.
